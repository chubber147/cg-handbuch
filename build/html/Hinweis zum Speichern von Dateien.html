Hinweis zum Speichern von Dateien¶

Wie bisher auch verfügen alle Benutzer über Speicherplatz auf dem Server. Egal an welchem Rechner in den Computerräumen oder in 210 Sie sich anmelden; bei der erfolgreicher Anmeldung wird dieses Verzeichnis mit dem Rechner, an dem sie arbeiten, verknüpft. Nur in dieses Verzeichnis sollten Sie ihre Dateien speichern („Home_auf_Server”). Sie finden einen entsprechenden Link auf dem Desktop der Ubunturechner. Ebenso finden Sie einen Link auf dem Desktop des Windowsrechner, der Sie zu ihrem Homeverzeichnis führt.

Wichtig

Wenn Sie ihre Dateien anderswo speichern, werden diese nicht auf ihrem Home_auf_Server-Verzeichnis gespeichert. Bitte erinnern Sie auch die Schüler daran, denn diese vergessen das immer mal wieder.
